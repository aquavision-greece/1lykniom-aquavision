# AquaVision by AquaMinds

# Περιγραφή
Το AquaVision είναι μια καινοτόμα συσκευή που ασχολείται με την ποιότητα του νερού. Αποτελείται από 2 τομείς: 
1) Αναλύει τη ποιότητα του νερού με τη βοήθεια ποικίλων αισθητήρων,
2) έπειτα φιλτράρει το νερό χρησιμοποιώντας ένα αυτοσχέδιο αλλά αποτελεσματικό φίλτρο.

# Μέρη του συστήματος
Αισθητήρες ανάλυσης νερού:
- pH sensor
- ORP sensor
- TDS sensor
- Turbidity sensor (θολότητα)
- Θερμόμετρο

Το φίλτρο
  Το φίλτρο είναι φτιαγμένο από ένα απλό πλαστικό δοχείο, που μέσα περιέχει σταδιακά προς τα κάτω:
  - βαμβάκι/γάζα (φίλτρο ιζήματος)
  - καθαρή λεπτόκκοκη ποταμίσια άμμος
  - φίλτρο ενεργού άνθρακα

# Παιγαγωγικοί στόχοι
Οι παιδαγωγικοί στόχοι του project είναι:

- Να κατανοήσουν οι μαθητές βασικές παραμέτρους ποιότητας του νερού (pH, ORP, TDS, θολότητα, θερμοκρασία) και τον ρόλο τους στην περιβαλλοντική ισορροπία και τη δημόσια υγεία.

- Να εξοικειωθούν με τη χρήση, τη λειτουργία και τη σωστή αξιοποίηση αισθητήρων σε ρομποτικά συστήματα μέτρησης φυσικοχημικών μεγεθών.

- Να εφαρμόσουν τη διεπιστημονική προσέγγιση STEM.

- Να αναπτύξουν δεξιότητες σχεδιασμού ενός συστήματος φιλτραρίσματος νερού.

- Να καλλιεργήσουν δεξιότητες επίλυσης προβλημάτων, κριτικής σκέψης και πειραματικής διερεύνησης.

- Να ενισχύσουν την περιβαλλοντική τους ευαισθητοποίηση.
